# Customer Login Smoke Test Cases

| Test Case ID | Test Case Title | Preconditions | Test Steps | Expected Result |
| :--- | :--- | :--- | :--- | :--- |
| **ST-01** | **Verify Successful Login** | 1. User is on the Login page.<br>2. A valid registered account exists. | 1. Enter a valid **Username**.<br>2. Enter a valid **Password**.<br>3. Click the **LOG IN** button. | 3. User is successfully logged in and redirected to the "Account Overview" page. |
| **ST-02** | **Verify Navigation to Registration** | 1. User is on the Login page. | 1. Click on the **Register** link. | 1. User is redirected to the "Signing up is easy!" registration form page. |
| **ST-03** | **Verify Navigation to Forgot Login Info** | 1. User is on the Login page. | 1. Click on the **Forgot login info?** link. | 1. User is redirected to the "Customer Lookup" page. |
